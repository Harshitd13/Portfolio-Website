# Portfolio Website with Flask

## Objective

A simple personal portfolio website built using Flask.

## Features

- Flask routing
- Personal information section
- Skills section
- Contact form

## Technologies Used

- Python
- Flask
- HTML
- CSS

## Project Structure

- app.py
- templates/index.html
- static/style.css

## How to Run

1. Install Flask:

   pip install flask

2. Run the application:

   python app.py

3. Open your browser and visit:

   http://127.0.0.1:5000
